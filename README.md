# 1-Space Website - College Assignment

This is my submission for the 2nd semester web design assignment. We were given a Figma design by our lecturer and had to recreate it as a webpage.

## About the Assignment

Our task was to convert a Figma design into a working HTML/CSS website. The design was for "1-Space" - a coworking space company. We had to match the design as closely as possible.

## What's in the Design

The Figma design had these sections:

- Navigation bar with logo and menu items
- Hero section with background circles
- Services section
- Vision and mission
- Benefits/features
- Photo gallery
- Call-to-action section
- Footer with links

## How I Built It

- Used the Figma file to get exact pixel measurements
- Wrote HTML for the structure
- Used CSS for styling and positioning
- Added fonts from Google Fonts (Montserrat, Poppins)
- Used all the images from the design
- Added a script.js file (empty for now - maybe will add JavaScript later)

## Challenges

- Getting every element in the right position was tough
- Had to carefully copy the spacing from Figma
- Took time to match colors and font sizes

## What I Learned

- How to use Figma for web design
- Positioning elements with CSS
- Working with different font families
- Organizing CSS code

## Files Included

- `index.html` - The webpage
- `style.css` - All the styling
- `script.js` - Empty file (haven't added JavaScript yet)
- `images/` - All the design assets
